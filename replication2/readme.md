# Задание 1

```
Опишите основные преимущества использования масштабирования методами:

1. активный master-сервер и пассивный репликационный slave-сервер;
2. master-сервер и несколько slave-серверов;
3. активный сервер со специальным механизмом репликации — distributed replicated block device (DRBD);
4. SAN-кластер.
```
```
1. В случае сбоя master-сервера, slave-сервер может быть быстро переведен в активное состояние,
обеспечивая минимальное время простоя. Некоторые операции чтения могут быть перенесены на slave-сервер,
что снижает нагрузку на master-сервер и увеличивает производительность.

2. Чтения можно распределять между несколькими slave-серверами, что значительно снижает нагрузку на master-сервер и
повышает общую производительность системы. С несколькими slave-серверами система становится более устойчивой к сбоям,
так как потеря одного slave-сервера не приводит к полной недоступности реплик.Легко добавить новые slave-сервера
для увеличения производительности без существенных изменений в архитектуре/

3.DRBD обеспечивает синхронную репликацию данных на уровне блоков, что минимизирует риск потери данных при сбоях.
DRBD можно использовать с различными файловыми системами и операционными системами/

4. SAN-кластеры обеспечивают высокую доступность данных и
отказоустойчивость за счет использования резервированных путей доступа к данным и распределения нагрузки.
Несколько серверов могут иметь одновременный доступ к одному и тому же хранилищу,
что упрощает управление и масштабирование системы. 
```
